# Employee Management System

## Project Description

This is a console-based Employee Management System developed using Python and MySQL. It allows users to manage employee records efficiently through CRUD operations.

## Features

- Add Employee
- View Employees
- Search Employee by ID
- Search Employee by Name
- Update Employee Details
- Delete Employee
- Employee Statistics Dashboard
- Input Validation
- Exception Handling
- Duplicate Employee ID Check

## Technologies Used

- Python
- MySQL
- mysql-connector-python
- Visual Studio Code

## Database

The project uses MySQL as the backend database.

## How to Run

1. Install Python
2. Install MySQL
3. Install mysql-connector-python
4. Create the database and employees table
5. Run:

```bash
python main.py
```
